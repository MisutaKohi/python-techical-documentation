Intro

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • Web Development (server-side)
  • Software Development
  • Mathematics
  • System Scripting
Syntax

Python syntax can be executed by writing directly in the Command Line:

>>> print("Hello, World!") Hello, World!

Or by creating a python file on the server, using the .py file extension, and running it in the Command Line:

C:\Users\Your Name>python myfile.py
Comments

Comments can be used to explain code, make it more readable, or even to prevent a line from executing when testing.

# this is a single line comment """ This is a multi-line comment. """
Variables

Python has no command for declaring a variable.

A variable is created the moment you first assign a value to it.

x = 5 # int variable y = "John" # string variable
Data Types

Variables can store data of different types, and different types can do different things.

Python has the following data types built-in by default:

  • Text Types: str
  • Numeric: int, float, complex
  • Sequence: list, tuple, range
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview
  • None: NoneType

You can get the data type of any object using the type() function.

In Python, the data type is set wehn you assign a value to a variable.
